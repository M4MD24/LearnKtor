## <span style="font-style: italic; color: #FF1493;">Tips for search important comments</span>

- For Search in Intellij IDEA `CTRL + Shift + F`, Select (In Project Option), and:

    - Search `! */` for **<span style="color: #FF1493;">Advices, Tips and Errors</span>**
    - Search `. */` for **<span style="color: #FF1493;">Definitions</span>**
    - Search `/* ~` for **<span style="color: #FF1493;">The Difference Between</span>**
    - Search `/* ↓` for **<span style="color: #FF1493;">Examples</span>**
    - Search `/* √` for **<span style="color: #FF1493;">Similar Concepts and Definition of Letters</span>**
    - Search `/* ■` for **<span style="color: #FF1493;">Tables</span>**
    - Search `// •` for **<span style="color: #FF1493;">Important Things</span>**
    - Search `// >` for **<span style="color: #FF1493;">Another Solution</span>**
    - Search `[//]: # (Notes)` for **<span style="color: #FF1493;">Important Notes</span>**
    - Search `[//]: # (Notes How)` for **<span style="color: #FF1493;">Notes How</span>**
